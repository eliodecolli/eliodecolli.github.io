Basic CQRS implementation

This project was a fun task I had to do on my spare time to learn more about the CQRS pattern and event sourcing. The real fun part was learning about RabbitMQ, data-consistency strategies as well as implementing my own in-memory cache solution.

What really surprised me, is that when I first heard about CQRS it made little to no sense in my mind as to why this kind of pattern is considered useful. However, as I went along with this project I started to see its importance, especially in a microservice architecture.
